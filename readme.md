# mytool

个人使用的工具箱包含4个工具：文件重命名、文件转 base64 编码、图片元数据获取、图片旋转标记修正

The toolbox for personal use contains 4 tools: file renaming, file conversion to base64 encoding, image metadata acquisition, image rotation mark correction

## 文件重命名

可以将文件夹内指定类型的文件进行重命名，本人主要用于AI图片数据集的重命名。

You can rename files of the specified type in the folder. I mainly use it for renaming AI image datasets.

## 文件转 base64 编码

可以将200kb以内文件转为 base64 编码，主要用于前端 css 中文件处理，如 font 和 img。

It can convert files within 200kb to base64 encoding, which is mainly used for file processing in front-end css, such as font and img.

## 图片元数据获取

可以获取图片的元数据信息。

The metadata information of the image can be obtained.

## 图片旋转标记修正

将精灵标注助手的异常方向图片修改为正常显示，若需更改标记时的数据记得选择json的导出数据信息。

Change the abnormal direction picture of the wizard marking assistant to normal display. If you need to change the data when marking, remember to select the json export data information.
